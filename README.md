### Jak odpalic
1. Pobieramy docker'a https://www.docker.com/
2. Idziemy do katalogu naszego projektu
3. Wpisujemy w wierszu poleceń komende ./startShop

### Jak zamknąć
1. Idziemy do katalogu naszego projektu
2. Wpisujemy w wierszu poleceń komende ./closeShop
3. **Jeżeli nie użyjemy tego skryptu przed commitem zmiany w bazie danych nie zostana zapisane**
